## 프로젝트 제목

    1. 제목: mini Cuphead
    2. 원본게임 제목: Cuphead
    3. 원본게임 설명: 여러 런앤건·보스 스테이지를 클리어하는 게임
                  보스 스테이지는 일반/비행기 2종류가 존재

## 게임 컨셉

    1. 게임 '컵헤드(Cuphead)'의 모작
    2. '우당탕탕 새장' 스테이지(비행기) 1페이즈를 구현
        → 장르: 횡스크롤 액션 슈팅
    3. 스테이지 보스를 쓰러트리는 게임
