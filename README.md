## 목표 변경(일정 변경)

|주차|구현항목|세부내용|
|:------:|:---:|:---:|
|5주차|중간점검|플레이어 이동방식 추가 ,1~4주차 미구현 항목 보완|
|6주차|보스|보스 2가지 패턴 구현|
|7주차|화면전환|시작·메인·종료 화면 전환, (추가구현)종료 시 랭크 띄우기|


#### 변경사유

    플레이어 이동을 계획대로 구현했지만,
    움직임이 부자연스러워 타겟이동 방식을 추가할 계획(전환 버튼도 추가)
        -> 기존 7주차 중간 점검과 일정을 교체, 5 6 주차는 1주씩 밀림
